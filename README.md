# 🏡Global-Airbnb-Performance-Dashboard

Dataset Link: https://mavenanalytics.io/data-playground/airbnb-listings-reviews
---

## 🚀 Project Summary

The **Global Airbnb Performance Dashboard** is an end-to-end data analytics project built in **Power BI** to analyze and visualize Airbnb listing data across multiple locations.

This dashboard transforms raw data into **actionable insights**, enabling stakeholders to understand **market demand, pricing strategies, and customer behavior** — all in one interactive experience. 

---

## 🎯 Problem Statement

In a highly competitive short-term rental market, hosts and analysts face challenges such as:
* Determine optimal pricing strategies
* Understand demand across locations
* Identify high-performing property types
* Track availability and occupancy trends

👉 This project addresses these gaps using **interactive analytics and visual storytelling**.

---

## 💡 Solution Approach

* Cleaned and transformed raw Airbnb data using **Power Query**
* Built dynamic KPIs using **DAX**
* Designed a **multi-page interactive dashboard**
* Enabled filtering and drill-down for deeper analysis
* Delivered insights through **intuitive visual storytelling**

---

### 🔹 Key Performance Indicators (KPIs)

* 📌 Total Listings
* 💰 Average Price
* 📅 Availability Rate
* ⭐ Average Review Score

---

### 🔹 Interactive Visuals

* 🌍 Geographic distribution of listings
* 💵 Pricing analysis across cities & property types
* 🏠 Room type comparison (Entire home vs Private vs Shared)
* 📈 Availability trends over time
* ⭐ Review & rating insights

---

## 🔍 Key Insights & Findings

* 📍 **Urban hubs dominate listings**, indicating concentrated demand
* 💰 **Entire homes/apartments command premium pricing**
* ⭐ **Highly rated properties show better occupancy trends**
* 📊 **Price variability depends heavily on location & property type**
* 📅 **Availability patterns suggest seasonal demand fluctuations**
